# 🔐 NumCipher — Number Guessing Game

A sleek, cyberpunk-inspired number guessing game built using **HTML, CSS, and JavaScript**.
Your mission: decrypt the hidden number between **1 and 100** within limited attempts.

---

## 🚀 Features

* 🎯 Random number generation (1–100)
* 🔢 Limited attempts system (10 tries)
* 📊 Real-time stats tracking:

  * Attempts left
  * Total guesses made
* 📜 Guess history with visual feedback
* ⚡ Instant feedback:

  * Too high / Too low / Correct
* 🎨 Futuristic UI with:

  * Neon glow effects
  * Animated scanlines
  * Responsive layout
* 🔁 Restart game anytime

---

## 🕹️ How to Play

1. Enter a number between **1 and 100**
2. Click **Submit** (or press Enter)
3. Follow hints:

   * ↑ Too low → Guess higher
   * ↓ Too high → Guess lower
4. Win by guessing correctly before attempts run out

## 🧠 Game Logic Overview

* Random number is generated at start
* Player inputs are validated
* Duplicate guesses are prevented
* Feedback updates dynamically
* Game ends when:

  * Player guesses correctly ✅
  * Attempts reach zero ❌
